# AVL Tree Visualizer 🌳

An interactive **AVL Tree Visualizer** built with **Java + JavaScript + HTML/CSS**.  
It supports **insertions and deletions** with automatic balancing (LL, RR, LR, RL rotations).

## 🚀 Features
- Insert nodes dynamically
- Delete nodes with rebalancing
- Visualize tree structure on canvas
- Clean UI with interactive controls
- Hosted live on GitHub Pages

## 🔗 Live Demo
👉 [AVL Tree Visualizer](https://pavankumar-2005.github.io/AVL-Tree-Visualizer/)

## 📂 Project Structure
- `java_version/` → Java backend implementation
- `web/` → Frontend (HTML, CSS, JS)
- `legacy_python/` → Archived Python version (optional)

## 🛠️ Tech Stack
- Java (AVL logic)
- JavaScript (frontend visualization)
- HTML + CSS (UI)
